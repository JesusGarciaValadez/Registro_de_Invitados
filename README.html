• LEEME * Para cambiar la configuración de la aplicación debes ir a los siguientes archivos y cambiar los setting para la conexión de la DB de tu máquina.

  • snippets/config/config.php .- Aquí debes cambiar la constante SITE_URL por la URL donde vas a copiar la aplicación.

  • snippets/db/connection.php .- Aquí hay que poner el nombre de la db, usuario y contraseña para que pueda conectarse.
  • snippets/lib/dbConnect.php .- Aquí hay que hacer lo mismo que el punto anterior.
